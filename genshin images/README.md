# Genshin Images

Учебное iOS-приложение на SwiftUI — викторина по персонажам Genshin Impact.

---

## О приложении

Приложение состоит из трёх экранов:

**Персонажи** — список персонажей с изображением и именем. При нажатии открывается карточка с подробным описанием.

**Викторина** — показывается описание персонажа, нужно угадать кто это из трёх вариантов. При правильном ответе летят конфетти, при неверном — подсвечивается правильный вариант. Вопросы идут по перемешанной колоде, чтобы не повторяться подряд. На экране играет фоновая музыка с плавным появлением и затуханием при переключении вкладок.

**Настройки** — переключение заголовков страниц, выбор любимого региона (меняет акцентный цвет по всему приложению), превью размера текста.
